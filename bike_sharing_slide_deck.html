Bike Sharing: Data Visualization Project¶

by Maria Cambalova¶

Investigation Overview¶

In this project, I wanted to explore how the bike sharing system is utilized over time and by which type of users. I focused on the trips' duration and the number of trips with respect to different time periods (year, month, day, day of week, hour), and user type (subscriber/casual customer).

Dataset Overview¶

The original data contained nearly 2.4 million records for years 2017-2018 (record = anonymized bike trip), and 14 features. I kept all records, but only a subset of the features. After data wrangling, which included data type conversions and engineering of new features from the existing ones, the final dataset contains 9 features:

  • start and end station names
  • user type
  • year, month, day, day of week, and hour when the bike trip started
  • duration of a trip

Distribution of the number of bike trips over months in 2017 and 2018¶

The highest number of bike trips happen from May to October 2018; winter and spring months are weaker. Since the Ford GoBike service started operating in summer 2017, the data is incomplete, and the number of trips peaks in September and October.

The number of trips increases from June to October 2017, then slightly decreases in November and December 2017. However, it begins to rise from January 2018 again.

Distribution of the number of trips during day time by day of week (2017 and 2018)¶

More detailed look into the trips distribution reveals that the bike usage is different for week days and weekends. From Monday to Tuesday, two peaks are visible: one in the morning at 7-9h and one in the afternoon at 16-18h. On weekends, the most trips happen between 11-17h. This pattern is true for both years 2017 and 2018.

# Define function to create a pivot table to visualize in a heatmap
def create_counts_pivot(data, rows, columns):
    ride_counts = data.groupby([rows, columns]).size()
    ride_counts = ride_counts.reset_index(name = 'count')
    ride_counts = ride_counts.pivot(index = rows, columns = columns, values = 'count')
    return ride_counts

Distribution of the number of trips during day time by day of week: Subscribers vs. customers¶

There is a difference between subscribers' and customers' trips distributions - while the subscribers' trips happen most often on week days, the casual customers' trips happen most often on weekends. However, customers use the bike sharing system also during the peak hours on work days.

Trip duration distribution during year and week: Subscribers vs. customers¶

Subscribers' trips tend to take approximately the same time irrespective of the day and month, and are shorter than customers' trips. In other words, longer trips including the weekend ones are made by casual customers in all months.

# Define function for log transformation
def log_trans(x, inverse = False):
    if not inverse:
        return np.log10(x)
    else:
        return np.power(10, x)

# Create box plots to show relationship ride duration - user type - month/week day - 2018
fig, axs = plt.subplots(1, 2, figsize = (15, 6))
fig.suptitle('Subscribers\' and customers\' trip duration over months and days of week (2018)', fontsize = 16)

tick_locs = [1, 2, 5, 10, 20, 50, 100, 200, 500, 1000, 2000]

# Left - month
ax1 = sb.boxplot(data = bikes[bikes['year'] == 2018], x = 'month', y = 'duration_log', 
                 ax = axs[0], hue = 'user_type', palette = 'Blues')
ax1.set_title('Months', fontsize = 14)
ax1.set_xlabel('Month', fontsize = 14)
ax1.set_ylabel('Duration (minutes)', fontsize = 14)
ax1.set_yticks(log_trans(tick_locs), tick_locs)
ax1.get_legend().remove()

# Right - week day
ax2 = sb.boxplot(data = bikes[bikes['year'] == 2018], x = 'weekday', y = 'duration_log', 
                 ax = axs[1], hue = 'user_type', palette = 'Blues', width = 0.6)
ax2.set_title('Days of weeks', fontsize = 14)
ax2.set_xlabel('Day of week', fontsize = 14)
ax2.set_ylabel('')
ax2.set_yticks(log_trans(tick_locs), tick_locs)
ax2.set_ylim(ax1.get_ylim())

# Rotate x-axis labels
for ax in axs:
    plt.setp(ax.get_xticklabels(), rotation = 45)

# Set y-axis tick labels
plt.setp(axs, yticks = log_trans(tick_locs), yticklabels = tick_locs)

# Set legend
plt.legend(title = 'User type', bbox_to_anchor=(1.35, 1.0), title_fontsize = 14, fontsize = 12, frameon = False);
